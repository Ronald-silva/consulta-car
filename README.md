# Consulta Car

Gerenciador de veículos e CNH para consultas rápidas (IPVA, licenciamento, multas e serviços da CNH), sem precisar redigitar dados longos. Os dados ficam armazenados localmente no navegador.

## Rodar localmente

**Requisitos:** Node.js

1. Instalar dependências: `npm install`
2. Subir o ambiente de desenvolvimento: `npm run dev`

A aplicação sobe em `http://localhost:3000` (porta definida no script `dev`).
